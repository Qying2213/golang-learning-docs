# 练手项目列表

## 项目概览

| 项目 | 难度 | 预计时间 | 核心知识点 |
|------|------|----------|------------|
| HTTP 框架 | ⭐⭐⭐ | 2-3 周 | 路由、中间件、Context |
| 连接池 | ⭐⭐⭐ | 1-2 周 | 并发、资源管理、超时 |
| RPC 框架 | ⭐⭐⭐⭐ | 3-4 周 | 网络、序列化、反射 |
| 分布式缓存 | ⭐⭐⭐⭐ | 3-4 周 | LRU、一致性哈希、singleflight |
| 任务调度器 | ⭐⭐⭐ | 2-3 周 | 定时任务、并发控制 |

## 推荐学习顺序

1. **HTTP 框架** - 最基础，帮助理解 Web 开发
2. **连接池** - 理解资源管理和并发
3. **任务调度器** - 巩固并发和定时任务
4. **分布式缓存** - 进入分布式领域
5. **RPC 框架** - 综合性最强

## 学习建议

### 每个项目的学习步骤

1. **阅读 README**
   - 理解项目目标
   - 了解功能要求
   - 熟悉 API 设计

2. **先实现最小可用版本**
   - 不要一开始就追求完美
   - 先让代码跑起来
   - 逐步添加功能

3. **写测试**
   - 每完成一个功能就写测试
   - 使用表驱动测试
   - 测试边界情况

4. **阅读参考实现**
   - 对比自己的实现
   - 学习优秀的设计
   - 理解为什么这样设计

5. **优化和重构**
   - 使用 pprof 分析性能
   - 重构代码结构
   - 添加文档和注释
