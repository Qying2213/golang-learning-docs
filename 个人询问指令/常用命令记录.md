## 1. Ubuntu 连接 GitHub（SSH 方式）

### 1.1 生成 SSH 密钥
```bash
ssh-keygen -t ed25519 -C "你的邮箱@example.com"
```
- `-t ed25519`：密钥类型，ed25519 是目前最安全的算法
- `-C "邮箱"`：注释，用来标识这个密钥是谁的
- 注意是大写 `-C`，不是小写 `-c`

### 1.2 查看公钥
```bash
cat ~/.ssh/id_ed25519.pub
```

### 1.3 添加到 GitHub
1. 打开 https://github.com/settings/keys
2. 点击 "New SSH key"
3. 粘贴公钥，保存

### 1.4 测试连接
```bash
ssh -T git@github.com
```
看到 `Hi xxx! You've been authenticated` 就成功了

### 1.5 配置 Git 用户信息
```bash
git config --global user.name "你的名字"
git config --global user.email "你的邮箱@example.com"
```

### 1.6 克隆仓库
```bash
git clone git@github.com:用户名/仓库名.git
```

## 2. Docker 常用命令

### 2.1 镜像操作
```bash
# 查看所有镜像
docker images

# 拉取镜像
docker pull nginx

# 构建镜像
docker build -t 镜像名 .

# 删除镜像
docker rmi 镜像名

# 查看镜像详情（注意要带 tag）
docker inspect nginx:alpine
```

### 2.2 容器操作
```bash
# 运行容器
docker run -d --name 容器名 镜像名

# 查看运行中的容器
docker ps

# 查看所有容器
docker ps -a

# 停止容器
docker stop 容器名

# 删除容器
docker rm 容器名

# 进入容器
docker exec -it 容器名 sh
```

### 2.3 清理命令
```bash
# 删除悬空镜像
docker image prune

# 清理所有未使用的资源
docker system prune
```

## 3. 进程管理

### 3.1 查找进程
```bash
# 查找包含关键词的进程
ps aux | grep 关键词

# 排除 grep 自己
ps aux | grep 关键词 | grep -v grep

# 或者用中括号技巧
ps aux | grep [关]键词
```

### 3.2 杀死进程
```bash
# 正常终止
kill PID

# 强制终止
kill -9 PID
```

## 4. 文件操作

### 4.1 echo 重定向
```bash
# 覆盖写入（原内容会丢失）
echo "hello" > file.txt

# 追加写入（原内容保留）
echo "hello" >> file.txt
```

### 4.2 scp 远程复制
```bash
# 上传文件到服务器
scp 本地文件 用户名@服务器:远程路径

# 例子
scp main.go qinyang@192.168.139.16:~/

# 上传目录
scp -r 目录 用户名@服务器:远程路径

# 从服务器下载
scp 用户名@服务器:远程文件 本地路径
```

## 5. Nginx

### 5.1 启动/停止
```bash
# 启动
nginx

# 停止
nginx -s stop

# 重新加载配置
nginx -s reload

# 检查配置是否正确
nginx -t
```
