## ✅ 学习检验标准

### Linux 实操
- [ ] 能熟练使用基础命令（ls, cd, cp, mv, rm, cat, vim）
- [ ] 能查看和管理进程（ps, top, kill）
- [ ] 能查看网络状态（netstat, ss, curl）
- [ ] 能部署一个 Go 项目到服务器

### 进程与线程
- [ ] 能解释进程和线程的区别
- [ ] 能解释 goroutine 和线程的区别
- [ ] 知道常见的进程间通信方式
- [ ] 理解进程调度的基本原理

### 内存管理
- [ ] 理解虚拟内存和物理内存的关系
- [ ] 理解分页机制
- [ ] 了解 Go 的内存分配器

### 并发与同步
- [ ] 能正确使用 Go 的 Mutex、RWMutex
- [ ] 理解死锁的四个条件
- [ ] 能写出线程安全的代码

### I/O 模型
- [ ] 能解释阻塞 I/O 和非阻塞 I/O 的区别
- [ ] 理解 I/O 多路复用（select/poll/epoll）
- [ ] 理解 Go 的网络模型

## 💡 学习建议

1. **边学边练**：每学一个概念，写 Go 代码验证
2. **结合实际**：把概念和 Go 的实现对应起来
3. **重点突破**：进程/线程、锁、I/O 模型是面试重点
4. **不求甚解**：理解概念即可，不需要深入内核源码

## 🔗 与 Go 的关联

| OS 概念 | Go 对应 |
|---------|---------|
| 进程 | `os/exec` 包 |
| 线程 | `runtime` 中的 M |
| 协程 | goroutine (G) |
| 锁 | `sync.Mutex` |
| 信号量 | `chan` |
| I/O 多路复用 | `net` 包底层使用 epoll |
| 内存管理 | Go 内存分配器、GC |

学完这套教程，你会对 Go 的底层原理有更深的理解！
