```
## 💡 使用技巧补充

### 让输出更完整
在 Prompt 末尾添加：
```

请给出完整的代码，不要省略任何部分。

```

### 让输出更详细
```

请详细解释每一步的原因。

```

### 控制输出长度
```

请控制在 1000 字以内。

```

### 指定输出格式
```

请用 Markdown 格式输出，包含代码块。

```

### 要求分步骤
```

请分步骤详细说明，每步都给出示例。

```

---

> 💡 **提示**：好的 Prompt = 明确的目标 + 详细的背景 + 具体的要求 + 期望的格式
```
