## 📖 一、技术学习笔记模板

### 🔹 Prompt：系统学习某个技术

```
请帮我系统学习【Go 语言并发编程】，生成一份完整的学习笔记，包括：

1. 概念介绍
   - 什么是并发 vs 并行
   - Go 并发的核心理念

2. 核心知识点（每个知识点要有）
   - 概念解释
   - 代码示例
   - 使用场景
   - 注意事项

3. 知识点清单
   - Goroutine
   - Channel
   - Select
   - sync 包（Mutex、WaitGroup、Once 等）
   - Context
   - 并发模式（生产者消费者、工作池等）

4. 常见面试题（5-10道）
   - 问题 + 答案

5. 推荐学习资源

请用 Markdown 格式输出，适合保存为笔记。
```

### 🔹 Prompt：快速掌握新技术/框架

```
我需要快速学习【Elasticsearch】，请帮我生成一份快速入门指南：

1. 这是什么？解决什么问题？（100字内）
2. 核心概念解释（索引、文档、mapping 等）

6. 踩坑注意事项
7. 进阶学习路线

控制在 2000 字以内，突出实用性。
```

## 📝 二、项目经验总结模板

### 🔹 Prompt：项目复盘总结

```
请帮我对【智能人才平台】项目进行复盘总结，生成一份项目经验文档：

项目基本信息：
- 项目类型：本科毕业设计
- 技术栈：Go + Gin + MySQL + Redis + Vue3
- 开发周期：X 周
- 核心功能：用户管理、简历管理、职位发布、智能推荐

请帮我总结：

1. 项目概述（200字）
   - 项目背景
   - 解决的问题
   - 核心亮点

2. 技术架构
   - 整体架构设计
   - 技术选型理由
   - 各层职责说明

3. 核心功能实现
   - 功能点1：实现思路 + 核心代码
   - 功能点2：...
   - 功能点3：...

4. 技术难点与解决方案
   - 难点1：问题描述 → 解决思路 → 具体实现
   - 难点2：...

5. 项目收获
   - 技术成长
   - 经验教训

6. 待优化项
   - 后续可以改进的地方

这份文档将用于面试时讲解项目，请突出技术深度。
```
