## 🎭 四、模拟面试

### 🔹 Prompt：让 Claude 当面试官

```
请你扮演一位资深后端面试官，对我进行一场 30 分钟的模拟面试。

我的背景：
- 应届本科生
- 求职方向：Go 后端开发
- 做过的项目：智能人才平台（Go+Gin+MySQL+Redis+Vue3）

面试流程：
1. 先让我自我介绍
2. 然后问项目相关问题（3-4个）
3. 接着问 Go 语言问题（3-4个）
4. 再问数据库问题（2-3个）
5. 最后问一个系统设计问题

一次只问一个问题，等我回答后再问下一个。
每个问题可以有追问。
最后给我面试反馈和改进建议。

现在开始，请让我自我介绍。
```

## 💡 面试技巧总结

### 回答技术问题的 STAR 框架

```
Situation - 场景/背景
Task - 具体任务
Action - 我采取的行动
Result - 最终结果
```

### 不会的问题怎么办

```
"这个问题我之前了解得不够深入，但根据我的理解..."
"这块我还没有实践经验，但我理解的原理是..."
"可以允许我思考一下吗？"
```

### 反问环节的高质量问题

```
1. "团队目前在做的最有挑战性的技术难题是什么？"
2. "如果我入职，最开始会负责哪块工作？"
3. "团队的技术栈演进方向是什么？"
4. "团队的代码 review 和技术分享是怎么进行的？"
```
