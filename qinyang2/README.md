## 🚀 开始你的旅程

**第一步：** 阅读 [01-初级开发.md](./01-初级开发.md)

**记住三个原则：**

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   1. 先深后广                                                               │
│      先把 Go 语言吃透，再扩展其他技术栈                                       │
│                                                                             │
│   2. 以战代练                                                               │
│      学了就要用，做项目是最好的学习方式                                       │
│                                                                             │
│   3. 持续输出                                                               │
│      写博客、做分享，输出倒逼输入                                            │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
