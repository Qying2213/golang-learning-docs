## 项目学习建议

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   1. 循序渐进                                                               │
│      按阶段做项目，不要跳级                                                  │
│                                                                             │
│   2. 先模仿后创新                                                           │
│      先参考开源项目，理解后再自己实现                                         │
│                                                                             │
│   3. 注重质量                                                               │
│      代码规范、单元测试、文档齐全                                            │
│                                                                             │
│   4. 持续迭代                                                               │
│      项目可以持续优化，不断加入新功能                                         │
│                                                                             │
│   5. 开源分享                                                               │
│      把项目放到 GitHub，写好 README                                          │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
